# 《不被钱绑架的日常》

一本面向一线城市年轻上班族的故事型理财书，通过真实场景和具体人物，传递实用的理财理念和生活方式。

## 项目介绍

本书以林晚为主角，讲述了她从被消费主义包围到逐步实现财务自由的转变过程。书中通过12个日常场景，融入了预算管理、消费习惯、紧急资金、被动收入等理财核心概念，强调通过小习惯的改变实现财务自由，而非追求一夜暴富。

## 核心观点

1. **区分需求与欲望**：学会识别真正的需求，避免冲动消费
2. **建立预算系统**：通过记账和预算，掌控个人财务
3. **紧急资金储备**：建立3-6个月生活费的紧急资金
4. **被动收入建设**：通过指数基金定投等方式，逐步建立被动收入
5. **时间价值意识**：认识到时间是最宝贵的资产
6. **生活质量平衡**：在理财的同时，保持生活的品质和乐趣

## 目标读者

- 一线城市年轻上班族
- 对理财感兴趣但不知从何开始的人群
- 希望改善消费习惯、实现财务自由的年轻人

## 如何使用

1. 按照章节顺序阅读，跟随林晚的故事和成长
2. 结合自身情况，思考书中的理财理念如何应用到实际生活
3. 可以将书中提到的方法和工具，如预算模板、记账方式等，应用到个人财务管理中
